Project 1: YouTube Clone Website (Layout)

This project lays out the structure for a YouTube-inspired website. At the top, a header section contains the YouTube logo, a search bar with search and voice search icons, and icons for content uploading and notifications. A sidebar navigation menu shows the different sections of YouTube like Home, Shorts, Subscriptions, and Your Library. The main section of the page shows video previews using CSS-grids , each featuring a thumbnail image, video title, channel name, view count, and upload date. These previews are clickable, leading to the corresponding video URLs. Overall, this layout provides users with familiar functionalities found on YouTube, content discovery, and interaction within the platform.

Project 2: Calculator Website

This project represents a simple calculator web page. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division, along with parentheses for grouping and percentage calculation. Users can input their calculations using buttons on the interface, and the result is displayed dynamically below the buttons. The design features various button styles to differentiate different types of operations and inputs.

Project 3: Todo List

This projects represents a simple Todo List web application. It presents a user interface with an input grid where users can input the name and due date of their tasks, along with an "Add" button to include the task in the list. The list of tasks is displayed below the input grid. The functionality for managing tasks, such as adding them to the list, is handled using JavaScript, while the visual styling is provided by an external CSS.
